Задача 1 (X_L7_T1)

Създайте клас Fibs, реализиращ протокола за итериране. Следният код трябва да може работи с вашия клас:
```
  fibs = Fibs()
  for f in fibs:
     if f > 1000:
        print(f)
        break
```

Задача 2 (X_L7_T2)
