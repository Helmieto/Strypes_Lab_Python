Памет (добавяне, извличане, изтриване):

**-m+** - Memory Plus  
**-m-** - Memory Minus  
**-mr** - Memory Recall  

**CE** - Clear Entry  
**AC** - All Clear

Основни математически операции  (+, -. \*, ÷)  
Коренуване
